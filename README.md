# HGeoKG

HGeoKG: A Hierarchical Geographic Knowledge Graph for Geographic Knowledge Reasoning

Dataset name: `HGeoKG-MHT-670K`

## Paper

The corresponding paper is available here:

[HGeoKG: A Hierarchical Geographic Knowledge Graph for Geographic Knowledge Reasoning](https://www.mdpi.com/2220-9964/14/1/18)

## Repository Overview

This repository contains the released HGeoKG data organized into `1` to `13` subdirectories.

Each subdirectory contains one or more `.txt` files with tab-separated triples.
